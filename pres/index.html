AssertJ

Fluent assertions for Java

http://assertj.org

Easy to use

  • Unique entry point for all assertions : assertThat
  • IDE friendly : discover assertions using IDE code completion
  • Look at javadoc for code examples
  • Useful error message
  • Possibility to chain assertions

assertj-core

Lot of assertions for JDK types:

  • string
  • collection / array / map
  • date
  • file
  • numbers
  • exceptions
  • ...

Example : code to test


 public class TolkienCharacter {
   // boilerplate getter omitted
   public int age;
   private String name;
   // Race is an Enum with a name property
   private Race race;
 }

 // Mr Frodo ! 
 TolkienCharacter frodo = new TolkienCharacter("Frodo", 33, HOBBIT);

 // ... and his friends from the Fellowship of the Ring
 List<TolkienCharacter> fellowshipOfTheRing = newArrayList(
   frodo, sam, merry, pippin, gandalf, legolas, gimli, aragorn, boromir);

Basic assertions example


 // unique entry point to get all assertThat methods
 import static org.assertj.core.api.Assertions.*;

 // common assertions
 assertThat(frodo.getName()).isEqualTo("Frodo");

 // you can chain assertions
 assertThat(frodo).isNotEqualTo(sauron)
                  .isIn(fellowshipOfTheRing);

 // String specific assertions
 assertThat(frodo.getName()).startsWith("Fro")
                            .endsWith("do")
                            .isEqualToIgnoringCase("frodo");

Collection and map assertions

 // collection assertions
 assertThat(fellowshipOfTheRing).hasSize(9)
                                .contains(frodo, sam)
                                .doesNotContain(sauron);

 // extract fellowshipOfTheRing characters name and check them
 assertThat(fellowshipOfTheRing).extracting("name")
                                .contains("Boromir", "Gandalf", "Frodo")
                                .doesNotContain("Sauron", "Elrond");

 // ringBearers : Map<Ring, TolkienCharacter>
 // initialized with the elves rings and the one ring bearers.
 assertThat(ringBearers).hasSize(4)
                        .contains(entry(valya, elrond), entry(narya, gandalf))
                        .doesNotContainEntry(oneRing, aragorn)
                        .containsKeys(nenya, narya, vilya, oneRing);

More examples available in assertj-examples project.

Custom assertions

Write assertions specific to YOUR classes

 // instead of :
 assertThat(frodo.getName()).isEqualto("Frodo");
 assertThat(frodo.age).isEqualto(33);
 assertThat(frodo.getRace()).isEqualto(Race.HOBBIT);

 // you write :
 assertThat(frodo).hasName("Frodo")
                  .hasAge(33)
                  .hasRace(Race.HOBBIT);
