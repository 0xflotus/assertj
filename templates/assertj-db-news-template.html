What's coming in the next AssertJ-DB release:

Javadoc : for versions 1.1.x

Improvements :
  • Add from(Calendar) and now() methods to DateValue, TimeValue and DateTimeValue
  • issue #15 : Add support of UUID columns type (Otoniel Isidoro)
  • Add isOfClass(Class) assertion method for value
  • issue #19 : Add the support of BLOB and CLOB
  • issue #22 : Add isEqualTo(Object) for the values. Add hasValues(Object...) for the columns. Add hasValues(Object) and hasValues(Object, Object) for the columns of the changes
  • issue #25 : Add containsValues(...) methods for the columns
Bug fixes :
  • issue #21 : Add possibility to pass a reference containing null value like parameter to isEqualTo() and isNotEqualTo()
  • issue #23 : Fix the support of Number (bug when the mapping is a Double instance)
